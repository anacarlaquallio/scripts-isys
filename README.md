# Scripts iSys

Este repositório contém um script para exclusão automática de artigos durante a etapa de filtragem dos resultados obtidos com a string de busca.

## Como utilizar

1. Instale a dependência necessária:

```bash
pip install bibtexparser
```

2. Clone o repositório:

```bash
git clone https://github.com/anacarlaquallio/scripts-isys
```

3. Execute o script informando o arquivo `.bib` como entrada:

```bash
python3 script.py papers.bib
```

## Saída

O script gera um arquivo `.csv` contendo os artigos filtrados.
